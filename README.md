Worship Culture Landing Page
A clean and modern landing page for a church worship event, built using only HTML and CSS.

## Features

* Hero section with event description
* Event details (date, time, location)
* Speaker cards with images
* Call-to-action section
* Responsive layout using Flexbox

##  Built With

* HTML5
* CSS3
